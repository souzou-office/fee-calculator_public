# 報酬・登録免許税 自動計算ツール

司法書士の報酬基準表に基づく報酬および登録免許税の自動計算ツール（汎用版）。
報酬テーブル・加算項目・事務所情報などは設定画面で編集でき、内容はブラウザ（localStorage）に保存されます。特定の事務所に依存しない構成です。

## 機能

- 所有権移転・保存・抵当権設定・根抵当権設定・抹消・住所変更に対応
- 報酬テーブル自動参照（設定有/無で自動切替）
- 土地・建物の課税標準額分離入力（売買時）
- 住宅用家屋証明書による軽減税率対応
- 登記事項証明書・登記情報の報酬/実費自動計算
- 追加項目（報酬＋立替金）の自由追加・並び替え
- ご請求明細の画面表示（登記報酬・登録免許税・実費を項目ごとに内訳表示）
- 報酬テーブル・加算項目のカスタマイズ（設定画面・localStorage保存）
- 必要書類一覧の作成（事務所情報を設定して PDF 出力。事務所情報は localStorage 保存）

## 開発

```bash
npm install
npm run dev
```

## デプロイ

`main` ブランチにpushすると GitHub Actions 経由で GitHub Pages に自動デプロイされます。

Settings → Pages → Source を **GitHub Actions** に設定してください。
